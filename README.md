Atomic Design:

Atomic design is a methodology for web and application development in which we emphasize the modularity of the design where we will try to create a design system rather than a page where we take cues from chemistry as it composed of five distinct stages working together to create interface and the five stages are
    1.Atoms
    2.Molecules
    3.Organisms
T   4.Templates
P   5.Pages
By following this methodology we can create a system which is maintainable as it should stand the test of time.
